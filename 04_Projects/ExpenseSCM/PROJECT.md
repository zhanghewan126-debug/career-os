\# 📝 Development Principles

3\. README 面向招聘方

4\. docs 面向技术细节

5\. images 保存所有 SVG、draw.io 图纸

6\. 所有内容保持统一命名规范

7\. 每完成一个模块进行 Review 后再进入下一阶段
