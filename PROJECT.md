## 🎯 Project Vision

CareerOS 是一套长期维护的职业操作系统（Career Operating System）。

目标不是仅完成一份简历，而是打造一套完整的职业品牌，包括：

- 黄金简历（ATS）
- GitHub 技术作品集
- Portfolio
- 企业级项目包装
- 面试知识库
- AI 辅助开发体系
- Developer Playbook
- Prompt Library
- Offer 投递管理
- 长期职业成长体系
